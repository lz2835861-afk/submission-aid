AI目录网站的共同规律：
1. 都需要一句话宣传语（60-80字符）+ 详细描述（250-500词）
2. 都需要分类标签（AI分类 / 标签）——选对分类决定曝光
3. 都需要定价模式（免费 / 免费增值 / 免费试用 / 付费）
4. 都需要核心功能列表（5-8个核心功能）
5. 都需要目标用户群体（谁在使用）
6. 竞品（Cursor/Claude/Copilot/Codex/Qoder/Trae）均已大量在列
7. 评论区/评价区域是重要参考——用户关心"实际效果""是否值得付费""和竞品比如何"

你的 AI 办公 Agent 套件 — 沉淀企业资产，赋能数字员工。

产品描述（约320词）

WorkBuddy 是腾讯打造的企业级 AI Agent 套件。它整合了一套精选的垂直领域 Agent，包括面向软件工程的 CodeBuddy、面向设计的 Ardot、面向产品与运营的 Miora，并构建于统一的 Agent 开发平台之上。该套件旨在帮助组织构建数字员工——即在集中管控下与人类团队 7×24 小时协同工作的 AI Agent。 平台提供智能规划与自主执行能力，能够理解自然语言指令，将复杂任务分解并交付可验证的结果。WorkBuddy 直接操作本地文件（包括 Excel、PDF 和图片），执行批量处理、数据清洗和文件整理。多个子 Agent 可并行协作处理不同任务，支持实时进度追踪，并生成文档、表格、图表和演示文稿等交付物。 WorkBuddy 支持跨主流协作平台的远程操控——包括 Slack、Microsoft Teams、WeCom、飞书/Lark、钉钉和QQ，用户可从移动端发消息远程指挥桌面端操作，并与 Notion、Canva、Google Workspace 等生产力与创意工具对接。定时任务能力支持 7×24 小时无需人工干预的自动化工作流。除了内置的 20+ 技能外，WorkBuddy 还可接入 SkillHub——一个汇聚 78,000+ 即用型技能的 AI 技能社区，并兼容 Open Claw 和 Claude 插件生态；团队管理、基于角色的授权以及内置的 GLM、Kimi、MiniMax 大语言模型共同完善了企业级能力栈。 部署选项包括 SaaS、VPC 专有云和完全本地私有化部署，满足金融、政务、医疗等行业的安全要求。

核心功能亮点（用于列表页展示）

  1. 垂直 Agent 套件 — CodeBuddy、Ardot、Miora 预配置覆盖工程、设计和产品工作流。
  2. 数字员工模式 — 7×24 小时 Agent，由人员或组织指挥调度。
  3. 企业管理平台 — Agent 在组织内可见、可控、可管理。
  4. 企业 AI 资产中心 — 知识库、技能、专家、连接器和大模型集中管理。
  5. 灵活部署 — SaaS、VPC 和本地部署选项，满足受监管行业需求。
  6. 行业解决方案 — 面向金融、投资、法律等领域量身定制。
  7. 跨能力统一 — Credits、OneID 和安全能力跨套件共享。
  8. Agent 与资产治理 — 统一的策略、审计和生命周期管控。

核心功能（详细版）

  1. 智能规划与执行 — 理解自然语言，思考并执行；自主拆解复杂任务，像专业同事一样交付可验证结果。
  2. 本地环境感知 — 直接操作本地 Excel、PDF、图片，进行批量处理、数据清洗和文件整理。
